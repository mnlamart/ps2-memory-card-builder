feat: initial implementation of PS2 Memory Card Manager

A web application for managing PlayStation 2 memory cards, built with React Router v7.

Features:
- Upload and manage PS2 memory card files (.ps2, .mc2)
- View memory card contents and save file details
- Import save files in various formats (.max, .psu, .sps, .xps, .cbs, .psv)
- Export save files in multiple formats
- Rename memory cards with user-friendly names
- Format and check memory card integrity
- Delete memory cards and individual save files
- Extract and view save file icons and metadata

Technical stack:
- React Router v7 with React 19
- TypeScript
- Tailwind CSS v4
- Vite for build tooling
- mymcplusplus for memory card operations
- Express server
- Playwright for E2E testing

Developer experience:
- React Grab integration for AI-assisted development
- Comprehensive error handling and user feedback
- Memory card metadata management system
- File upload and storage management
